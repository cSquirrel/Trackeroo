# Trackeroo

This repo contains versioned implementations of Trackeroo, each self-contained under its own top-level folder. The current implementation is [`v1/`](v1/).

**Before working on the current implementation, `cd` into `v1/`** and treat it as your working directory — the detailed project conventions live in [`v1/CLAUDE.md`](v1/CLAUDE.md) (Claude Code auto-loads the nearest `CLAUDE.md`, so working from inside `v1/` picks up the right instructions).

`.github/workflows/` stays at the repo root because GitHub only discovers workflows there; its job paths are prefixed into `v1/`.
